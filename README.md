# color-palette-app


## Install:

git clone https://github.com/makkoncept/colorpalette.git  
cd colorpalette
Create a virtualenv:

python3 -m venv venv   
Activate it on Linux:

. venv/bin/activate  
Or on Windows cmd:

venv\Scripts\activate.bat  
Install requirements:

pip install -r requirements.txt  
Run:

python run.py
View on localhost:5000
